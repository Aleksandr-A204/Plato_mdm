# Plato.MDM

```
Перечень справочников
    Отображение перечня справочников
        Свойства для отображения: Наименование, Предметная область, Уровень
    Просмотр детальной информации о справочнике
    Редактирование детальной информации о справочнике
    Добавление нового справочника
        Наименование - обязательное поле
    Удаление справочника
        Подтверждение при удалении
    Сортировка перечня справочников
        По одному из свойств: Наименование, Предметная область, Уровень
        По возрастанию/убыванию
    Поиск справочника по наименованию
    Фильтация перечня справочников
        По одному или несколиким свойствам: Предметная область, Уровень
        Множественный выбор значений для фильтрации внутри одного свойства (условие ИЛИ)
        Множественный выбор значений для фильтрации разных свойств (условие И)
        Пример фильтра: DomainId IN (ООПТ, Животные, Растения) AND LevelId IN (Регион)
    Отображение перечня версий по увыбранному справочнику

Версии справочника
    Отображение перечня версий по увыбранному справочнику 
        Свойства для отображения: Версия, Дата версии, Источник данных
        Список должен быть отсортирован в обратном порядке по свойству Дата версии
    Просмотр детальной информации о версии справочника
    Редактирование детальной информации о версии справочника 
    Добавление новой версии справочника
    Удаление версии справочника
        Подтверждение при удалении
    Отображение данных по выбранной версии справочника

Данные справочника    
    Отображение записей справочника по выбранной версии справочника
        Таблица с записей справочника указана в поле TableName
        Состав полей таблицы должен формироваться динамически в зависимости от труктуры таблицы TableName
        Если таблица содержит внешние ключи (FOREIGN KEY) то вместо ID должно отображаться наименование записи из связанной таблицы
    Сортировака записей справочника
    Поиск записей справочника по Наименованию
    Редактирование записей справочника
        Редактирование FOREIGN KEY полей путем выбора из выпадающего списка
        Сохранение или отмена изменений отдельными кнопками
    Добавление записей в справочник
    Удаление записей из справочника
        Множественный выбор записей
        Подтверждение при удалении
```
